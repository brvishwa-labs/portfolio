# Profile Image

Please add your profile photo here as `profile.jpg`

**Instructions:**
1. Save your profile photo as `profile.jpg` in this directory (`public/assets/images/`)
2. The image should be square (recommended: 400x400px or larger)
3. Supported formats: JPG, PNG, or WebP
4. The image will automatically display in the Hero section of your portfolio

**Note:** If no image is found, a placeholder emoji will be displayed instead.
